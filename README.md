
# Portafolio de Herramientas Digitales e IA

## Sección 1: Ofimática en la Nube y Local
En esta área he desarrollado habilidades para la creación y edición de documentos técnicos de alta calidad. El enfoque principal ha sido:

* **Microsoft Word**: Manejo avanzado de **estilos**, creación de **índices automáticos** y organización de documentos extensos.
* **Google Docs**: Uso de herramientas en la nube para el **trabajo colaborativo**, permitiendo la edición simultánea y el control de cambios entre múltiples usuarios.

## Sección 2: Gestión de Documentos
Para optimizar el flujo de trabajo profesional, utilizo **ILovePDF**, una herramienta esencial para la administración de archivos. Mis capacidades incluyen:

* **Unión y Compresión**: Combinar múltiples informes en un solo archivo y reducir su peso para facilitar el envío por plataformas institucionales.
* **Conversión**: Transformar documentos de diversos formatos a **PDF** para asegurar que el formato se mantenga intacto en cualquier dispositivo.

## Sección 3: Inteligencia Artificial Generativa
He integrado el uso de **ChatGPT** en mis procesos académicos para mejorar la documentación
técnica. La IA me permite:

* **Estructuración**: Definir esquemas lógicos para informes de sistemas y proyectos de hardware.
* **Redacción**: Refinar el tono profesional y la claridad de los textos técnicos, asegurando una comunicación efectiva de conceptos complejos.
